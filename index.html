Configuració de sync (JSONBin)
JSONBin és un servei gratuït que guarda les teves dades al núvol. Funciona com una base de dades simple, accessible des de qualsevol dispositiu.
1
Ves a jsonbin.io i crea un compte gratuït (botó "Sign Up")
2
Un cop dins, ves a API Keys al menú de l'esquerra → copia la teva Secret Key
3
Enganxa la clau aquí sota i fes clic a "Connectar" — la resta ho fa l'app automàticament.

ℹ️ Com funciona el sync

Cada vegada que guardes una tasca o registres hores, l'app puja les dades a JSONBin automàticament.

Al carregar l'app en qualsevol dispositiu, descarrega les dades del cloud. Així sempre tens la versió més recent.

Les dades també es guarden localment per si no tens connexió.
