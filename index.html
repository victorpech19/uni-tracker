🔄 Backup / Importar

Exporta una còpia completa (tasques + registres d'hores + calendari) com a fitxer JSON. Per importar tasques noves sense esborrar les existents, fes servir el botó "Importar tasques" de la pestanya Tasques.

Format JSON per importar tasques 
{
  "tasks": [
    {
      "subject": "Càlcul",
      "topic": "T3 - Integrals",
      "type": "Problemes",
      "detail": "Llista 4",
      "estimatedHours": 3,
      "status": ""
    }
  ]
}

Camps obligatoris: subject. Opcionals: topic, type, detail, estimatedHours, status.
Les tasques amb la mateixa assignatura + tema + tipus no es duplicaran.

ℹ️ Com funciona el sync

Cada vegada que guardes una tasca o registres hores, l'app puja les dades a JSONBin automàticament.

Al carregar l'app en qualsevol dispositiu, descarrega les dades del cloud. Així sempre tens la versió més recent.

Les dades també es guarden localment per si no tens connexió.
